\subsection{Verificare se dei vettori sono l.i.}
$v_1 = (1, -3, 7) \quad v_2 = (2, -1, -1) \quad v_3 = (-4, 2, 2)$

\(
\begin{cases}
	x + 2y -4z = 0 \\[-0.3em]
	-3x -y +2z = 0 \\[-0.3em]
	7x -y +2z = 0 \\
\end{cases}
\rightarrow
\begin{cases}
	x = 0 \\[-0.3em]
	y = 2z \\[-0.3em]
	0 = 0 \\
\end{cases}
\rightarrow
t\begin{bmatrix}
	0 \\[-0.3em]
	2 \\[-0.3em]
	1 \\
\end{bmatrix} \forall t \in \mathbb{R}
\) sono l.d.

$0v_1 + 2tv_2 + tv_3 = 0$

Se l'unica soluzione fosse stata $(0,0,0)$ allora sarebbero l.i.
